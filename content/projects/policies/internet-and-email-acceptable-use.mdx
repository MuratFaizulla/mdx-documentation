# Правила использования интернета и электронной почты

> **Интернет и электронная почта — только для учебной работы.**

---

## Обязанности учащихся

Учащиеся **обязаны**:

| Требование | Описание |
|-----------|--------|
| Соблюдать законы РК | Включая **Закон о компьютерных преступлениях**, **Закон о защите данных**, **Закон об авторском праве**, **Закон о телекоммуникациях** |
| Запрещено участие в незаконной деятельности | Нельзя **создавать, передавать или делиться** запрещённым контентом |
| Сообщать о проблемах | При обнаружении **незаблокированного вредоносного контента** (экстремизм, порнография) — **немедленно сообщить старшему сотруднику** |

---

## Запрещённые действия

Учащимся **запрещено**:

| Запрещено | Последствия |
|---------|-----------|
| Использовать **прокси-сайты** для обхода фильтров | Автоматическое обнаружение → блокировка |
| Скачивать **.exe, фильмы, музыку (mp3)** | Без разрешения — нарушение |
| Запускать **неавторизованные программы** | Риск вирусов → отключение |
| Использовать **P2P-сети** (торренты, файлообменники) | Запрещено |
| Совершать **покупки** в интернет-магазинах | Не для школы |
| Использовать **школьную почту для спама** | Блокировка аккаунта |

---

## Важные уведомления

> **Школа не несёт ответственности** за сбои в работе ИТ-оборудования (аппаратного или программного обеспечения) и **последующие убытки**.

- **Резервные копии** создаются **ежедневно**.
- **Пользователь сам отвечает** за **сохранение важных данных**.

> **Политика может быть изменена** в любое время.  
> Актуальная версия публикуется на **официальном сайте школы**.
